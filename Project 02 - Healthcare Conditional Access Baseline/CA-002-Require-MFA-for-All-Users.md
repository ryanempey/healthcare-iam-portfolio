# CA-002 – Require MFA for All Users

## Objective

Require multifactor authentication for all workforce users accessing Microsoft cloud resources.

## Business Justification

Password-based attacks remain one of the most common attack vectors in healthcare environments.

Enforcing MFA significantly reduces the risk of:

- Credential theft
- Password spraying
- Phishing attacks
- Unauthorized account access

## Assignments

### Users

Included:

- All Users

Excluded:

- BreakGlass01
- BreakGlass02

## Cloud Apps

All Cloud Apps

## Conditions

None

## Access Controls

Grant Access:

- Require Multifactor Authentication

## Policy State

Report-only

## Expected Outcome

All workforce users must successfully complete MFA before accessing Microsoft resources.

## Healthcare Considerations

Break Glass accounts remain excluded to ensure emergency access during outages or authentication failures.
