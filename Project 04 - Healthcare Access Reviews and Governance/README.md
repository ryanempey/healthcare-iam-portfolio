# Project 04 – Healthcare Access Reviews and Governance

## Executive Summary

This project demonstrates access review and governance processes for a simulated healthcare organization.

The goal is to show how IAM teams validate that users retain only the access required for their current role.

## Business Scenario

Front Range Health Partners must regularly review access to reduce excessive permissions, orphaned access, contractor risk, and privileged account exposure.

## Reviews Included

| Review | Purpose |
|---|---|
| Quarterly Access Review | Validate department access |
| Contractor Access Review | Review external/non-employee access |
| Privileged Access Review | Review admin and elevated roles |

## IAM Concepts Demonstrated

- Access reviews
- Manager certification
- Least privilege
- Privileged access governance
- Contractor access governance
- Audit readiness
- Healthcare compliance support
