# Project 03 – Healthcare Identity Lifecycle Management

## Executive Summary

This project demonstrates identity lifecycle management for a simulated healthcare organization.

The goal is to show how IAM teams manage user access through Joiner, Mover, and Leaver workflows.

## Business Scenario

Front Range Health Partners needs a repeatable process for managing workforce identities.

The IAM team must ensure that users receive the right access at the right time and lose access when it is no longer needed.

## Workflows Included

| Workflow | Purpose |
|---|---|
| Joiner | Provision access for a new employee |
| Mover | Update access after a department transfer |
| Leaver | Remove access after termination |

## IAM Concepts Demonstrated

- User provisioning
- Group-based access control
- Role-based access control
- Department-based access
- License management
- Account disablement
- Session revocation
- Least privilege
- Healthcare access governance
