# Project 05 – Healthcare RBAC Design

## Executive Summary

This project demonstrates a Role-Based Access Control design for a simulated healthcare organization.

The goal is to show how access can be assigned based on job role, department, and least-privilege principles.

## Business Scenario

Front Range Health Partners needs a structured RBAC model to reduce manual access assignment and prevent excessive permissions.

Access should be granted through groups rather than direct user assignments.

## Scope

Departments included:

- Radiology
- Nursing
- Physicians
- HIM
- Billing
- IT
- Contractors

## RBAC Principles

- Least privilege
- Group-based access
- Department-based access
- Role-based access
- Separation of duties
- Simplified onboarding and transfers
